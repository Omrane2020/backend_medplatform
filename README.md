# Admin Backend (Express + Sequelize)

Starter backend for the React admin components. Features:
- Authentication (JWT)
- CRUD for Users, Subscriptions, Secretaries, Settings, Activity Logs
- Sequelize models + MySQL (mysql2)

Instructions:
1. Copy `.env.example` to `.env` and fill database credentials.
2. Install dependencies:
   npm install
3. Run:
   npm run dev
